Este é o repositório do meu portfólio profissional, criado para destacar minha trajetória como Analista de Tecnologia da Informação, especialista em Infraestrutura, Green Software e Engenharia de Software Sustentável.

## 🌐 Acesse o site
**[francildogryphon.com.br](https://francildogryphon.com.br)**

## 🎯 Sobre o Projeto

O objetivo deste site é servir como um hub central para as minhas atuações:
- **Trajetória Profissional:** Mais de 20 anos na área de TI, com certificações em Metodologias Ágeis, DevOps, Forense Digital e MBA.
- **Sustentabilidade (Green Software):** Divulgação de conceitos de consumo consciente de recursos de TI através do código.
- **Produção de Conteúdo:** Links diretos para os meus artigos (DIO), vídeos (YouTube) e repositórios (GitHub).

## 🛠️ Tecnologias Utilizadas

Este portfólio foi construído mantendo o princípio de *Green Software* de ter um código enxuto, de alta performance e energeticamente eficiente:
- **HTML5 Semântico:** Estruturação leve e focada em SEO e acessibilidade.
- **CSS3 Vanilla / Flexbox & Grid:** Estilização moderna e responsiva sem dependência de bibliotecas ou frameworks pesados.
- **JavaScript Vanilla:** Lógica enxuta para interações de UI (animações suaves, navegação).
- **Deploy Automático:** Otimizado e publicado através da **Cloudflare Pages**, garantindo alta velocidade global e baixo impacto ecológico.

## 🏃 Como Rodar Localmente

Para visualizar o projeto em sua máquina local, basta clonar o repositório e abrir o arquivo `index.html` em qualquer navegador web moderno:

```bash
# Clonar o repositório
git clone https://github.com/francildogryphon/francildogryphon-site.git

# Acessar a pasta
cd francildogryphon-site
```
Abra o `index.html` no seu navegador de preferência ou utilize extenções como *Live Server* no VS Code.
